# Abhay Das Malikaveettil Portfolio

A responsive personal portfolio website built with HTML, CSS, and JavaScript to showcase my projects, skills, internships, and contact details.

## Features

- Multi-page portfolio with `Home`, `About`, `Skills`, `Projects`, and `Contact`
- Recruiter-friendly homepage with resume download and clear calls to action
- Dedicated projects page covering portfolio work, internship work, and Java projects
- Contact section with email, LinkedIn, GitHub, and resume access
- Responsive design for desktop and mobile screens
- Simple static deployment to Netlify or GitHub Pages

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Formspree for contact form handling

## Setup

1. Clone the repository.
2. Open the project folder.
3. Run the site locally by opening `index.html` in your browser.

## Deployment

### Netlify

1. Push the code to GitHub.
2. Import the repository into Netlify.
3. Deploy the site.

### GitHub Pages

1. Push the repository to GitHub.
2. Open repository settings.
3. Enable GitHub Pages for the main branch.
